# Utility Bill Management - Frontend

This is the frontend application for the **Utility Bill Management System**, allowing users to view and pay utility bills such as water, gas, electricity, and internet. Built with **React**, **Vite**, **Tailwind CSS**, **DaisyUI**, and enhanced with **SweetAlert** notifications.

## Live Demo

[https://utility-bill-management.firebaseapp.com](https://utility-bill-management.firebaseapp.com)

## Features

- View all available utility bills
- Filter bills by category (Electricity, Water, Gas, Internet)
- View detailed bill information
- Pay bills for the current month
- Responsive design for mobile and desktop
- SweetAlert notifications for payments and actions
- Styled with Tailwind CSS and DaisyUI components
- Integration with backend API hosted on **Vercel** or **Firebase**

## Tech Stack

- **React** - Frontend UI library
- **Vite** - Fast build tool for development
- **Tailwind CSS** - Utility-first CSS framework
- **DaisyUI** - Prebuilt Tailwind UI components
- **SweetAlert2** - Beautiful popup alerts
- **React Router** - Page navigation
- **React Helmet Async** - Manage page titles
- **Fetch API** - Backend API communication
